# todo-list
📝 لیست کارهای روزانه  ابزاری ساده برای مدیریت کارهایی که باید انجام بدی. میتونی کار جدید اضافه کنی، کنارش تیک بزنی یا پاکش کنی. کارهای انجام شده خط میخورن. با دکمه‌های فیلتر، میتونی فقط کارهای انجام شده یا نشده رو ببینی. همه چیز توی مرورگرت ذخیره میشه، حتی بعد بستن. ساخته شده با HTML، CSS و جاوااسکریپت.
